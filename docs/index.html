What it does

Import

Scan a copied 431960 Workshop folder and import supported projects into a private Mac library.

Play

Run MP4, MOV, M4V, local web, image, and supported scene.pkg wallpapers on the desktop layer.

Screen saver

Install the bundled macOS screen saver so supported video wallpapers can animate when the screen saver starts.

Install

Open the DMG, drag the app into Applications, then launch it from the menu bar. The download button uses the GitHub API to resolve the newest WorkshopWallpaperBridge-macOS-arm64.dmg asset automatically.

Use it

  1. Copy your Wallpaper Engine Workshop folder from Windows: steamapps/workshop/content/431960.
  2. Open Workshop Wallpaper Bridge Settings from the macOS menu bar.
  3. Choose the copied 431960 folder, scan it, then import the supported wallpapers.
  4. Select an imported wallpaper and click Play on Desktop.
  5. Turn on Animate Screen Saver if you want supported video wallpapers to run when the macOS screen saver starts.
